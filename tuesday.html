Light on notes today. We started by whiteboarding our pre-work Python problems. Each problem had a randomly selected person attached to it, and those primary problem solvers were given help by the others.

Lucas was assigned to question 5, which dealt with creating a function that would determine whether numbers in a list were consecutive and return True or False. I had problems with determining how to move from one index to another.

The solution makes sense.

# This is the solution Lucas came up with to iterate through the list
                            def is_consecutive(a_list): # define the function "is_consecutive"
                                a_list.sort() # sort the list; this is my addition
                                for i in range(1, len(a_list)): # for "i" ("index item") in the range from index 1 to the length of a_list
                                    if a_list[i] - 1 != a_list[i - 1]: 
                                    """if the index of "i", which starts at index 1, minus the integer 1 is not equal to a_list index of i - 1 index number"""
                                        return(False)
                                return(True) # put outside the for loop so that if nothing returns false
                            
                            print(is_consecutive([24, 22, 23]))
