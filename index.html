Command Line

In the Windows environment there is the PowerShell. For Git you can use the Git Bash app to use a command line like you would see in a Unix or Linux environment.

Frequently Used Commands

  • pwd

    • Gets information on current path

  • ls

    • Lists the files and folders in the current directory

  • cd

    • Change directory; used to move up or down a directory path into different directories

    • cd [folder name]

      • Use Tab key to auto-complete

    • cd ..

      • Move back one folder in the path; use "../.." to move up two folders in the path

  • code .

    • Opens Visual Studio Code app

Visual Studio Code

Once the Visual Studio Code app has opened, you can use the terminal inside the application. Use the keyboard shortcut ctrl + ~ or cmd + ~ to open the terminal.

Setting Up a Github Repository

Before using the command line to upload to your Github repository, it needs to know which repository to work with. Set up a Github repository with the following steps:

  1. Log in to Github
  2. Click the + button in the upper-right corner; select New Repository from the dropdown menu
  3. Name the repository and give it a description
  4. Click the Create Repository button
  5. Copy the URL of the repository by clicking the clipboard icon; this will be used in step 4 of the Git Terminal Commands instructions below

Git Terminal Commands

  1. git init

    2. This tells the computer to initiate a local repository; you must be in the working directory; the command line will create a .git file in that directory

  2. git add .

    3. This adds all the files and directories in the working directory to the git repository

  3. git commit -m "first commit"

    4. This creates the first git package to be uploaded

  4. git remote add origin [Github repository URL]

    5. This uploads the local repository to the Github repository

  5. git push -u origin master

    6. What exactly does this do?

Workflow – Visual Studio Code and Git

Use the following steps when working in Visual Studio Code and backing up to Github

  1. Use ctrl + s on Windows or CMD + S on OSX to save your work locally
  2. Enter the following in your command line
    1. git add .
    2. git commit -m "changes made"
    3. git push -u origin master
